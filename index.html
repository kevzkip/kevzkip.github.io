Projects

Featured · Final Year Project

LoRaWAN Smart Irrigation System

A two-node LoRa-based smart irrigation system built for remote agricultural environments. Uses ESP32 + SX1278 Ra-01 LoRa modules with DHT11 temperature/humidity sensing and resistive soil moisture sensing. Implements automatic dual-condition irrigation logic, manual override via Blynk IoT, and RSSI/SNR link quality monitoring.

Field tested to 800 m in urban NLOS conditions in Nairobi. Designed with LPWAN principles for low-power, long-range IoT data pipelines.
