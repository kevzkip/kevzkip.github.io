## Featured Project

**LoRaWAN Smart Irrigation System** — Final Year Project

A two-node LoRa-based smart irrigation system using ESP32 + SX1278 Ra-01 modules, DHT11 sensor, resistive soil moisture sensing, and Blynk IoT for remote override. Field tested to 800 m in urban NLOS conditions in Nairobi. Implements automatic dual-condition irrigation logic and RSSI/SNR link monitoring.

`ESP32` `LoRa / SX1278` `Blynk IoT` `C++` `LPWAN` `DHT11`
